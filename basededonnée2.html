Base de données – Projet AssoSport

La base de données est le cœur du projet : elle centralise toutes les informations sur les Adhérents, les inscriptions et les compétitions, de façon sécurisée et structurée.

  • Table adhérents : Nom, prénom, date de naissance, adresse, email, téléphone, date d’inscription, statut, historique des adhésions.
  • Table club : Nom du club, adresse, membres, administrateurs, gestion des droits d’accès (admin, coach, adhérent…).
  • Table compétitions: Nom, date, lieu, disciplines concernées, participants inscrits, résultats, classements.
  • Table disciplines: Nom, description, niveau, compétitions associées.
  • Table inscriptions: Adhérent inscrit, compétition concernée, date d’inscription, statut (validée, en attente, refusée).
  • Table users: Identifiant, mot de passe (sécurisé), rôle attribué (admin, coach, adhérent…), historique de connexion.

Points clés

  • Sécurité : Accès restreint, gestion fine des permissions, protection des données sensibles.
  • Performance : Requêtes optimisées, affichage rapide, expérience fluide pour l’utilisateur.
  • Évolutivité : Structure flexible, ajout facile de nouvelles tables ou fonctionnalités selon les besoins.
